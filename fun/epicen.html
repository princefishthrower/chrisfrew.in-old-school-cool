Numerical
Tracers

Meine M.S. Theses. Wir behandeln Moleküle als Graphen (Atomen als Knoten und Anleihen als Kanten) und anwenden ein "constrained hierarchical pattern matching algorithm" (CHPMA, eingeschränkt hierarchische Mustervergleichsalgorithmus auf Deutsch) zu bestimmen die Symmetrien der Struktur der Moleküle in jeder Reaktion, letztlich die Bestimmung der Masse Mengen jeder Position in jedem Molekül an jedem Zeitschritt der Verbrennung. (***Notiz: das Repository ist selbst privat; der Link öffnet sich ein Bildschirmfoto des Repo als zeigen und erzählen.)

UPursuit Event Organizer
Site Build

An event organizer that leverages both the Dark Sky Forecast and Google Geocoding APIs with real-time table sorting thanks to List.js. The distance from each event to the user's location is also calculated.
