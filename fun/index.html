Projects

The following are links to a few of my projects:


Numerical
Tracers

My M.S. thesis. We treat molecules as graphs (atoms as nodes and bonds as edges) and apply a hierarchical pattern matching algorithm to determine the symmetries of structures of the molecules in each reaction, ultimately determining the mass amounts of each location in each molecule at every timestep of combustion. (***Note: the repo itself is private; the link brings up a screenshot of the repo as a show and tell.)

UPursuit Event Organizer
Site Build

An event organizer that leverages both the Dark Sky Forecast and Google Geocoding APIs with real-time table sorting thanks to List.js. The distance from each event to the user's location is also calculated.



I've also done some backend database work employing both webscraping and formatting data retrieved from two popular housing APIs - Zilyo (for airbnb) and the Expedia Affiliate Network (for hotels).
